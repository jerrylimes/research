# Errata 4980 - RFC 7483

- **RFC Title:** JSON Responses for the Registration Data Access Protocol (RDAP)
- **RFC Publication Date:** March 2015

```
Section 5.5 says:


country -- a string containing the name of the two-character

country code of the autnum

// state how to fix the above text here

```

## Issue description

The description of the `country` field in Section 5.5 is inconsistent with its description in Section 3. This inconsistency could lead to confusion for implementers and users of the RDAP protocol.

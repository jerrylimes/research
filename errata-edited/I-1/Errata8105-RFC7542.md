# Errata 8105 - RFC 7542

- **RFC Title:** The Network Access Identifier
- **RFC Publication Date:** May 2015

```
Section 3.4 says:


Examples of valid Network Access Identifiers include the following:

[...]

        \(user\)@example.net

// state how to fix the above text here

```

## Issue description

The given NAI example does not match with its description. This inconsistency between the examples and the ABNF rules could lead to incorrect implementations. The inconsistency needs to be resolved either by correcting the example or updating the ABNF rules.

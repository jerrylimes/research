# Errata 5324 - RFC 7599

- **RFC Title:** Mapping of Address and Port using Translation (MAP-T)
- **RFC Publication Date:** July 2015

```
Section Appendix A says:


Example 5:



PSID:                    0x20 (provisioned with DHCPv6)

// state how to fix the above text here

```

## Issue description

The original example uses a PSID value that does not match the provided IPv6 address and port ranges. This inconsistency would lead to confusion in interpreting the example and implementing the specification correctly.

# Errata 5765 - RFC 7170

- **RFC Title:** Tunnel Extensible Authentication Protocol (TEAP) Version 1
- **RFC Publication Date:** May 2014

```
Section 4.2.2 says:


   M



      Mandatory, set to one (1)

// state how to fix the above text here

```

## Explanation

The requirement in Section 4.2.2 presents a contradition to the mandate of Section 4.3.1. This directly impacts implementation as it provides conflicting instructions on how to handle this specific TLV.

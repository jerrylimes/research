# Errata 4252 - RFC 7230

- **RFC Title:** Hypertext Transfer Protocol (HTTP/1.1): Message Syntax and Routing
- **RFC Publication Date:** June 2014

```
Section 2.7.2. says:


     https-URI = "https:" "//" authority path-abempty [ "?" query ]

                 [ "#" fragment ]

// state how to fix the above text here

Notes:

See erratum 4251 on the same document.
```

## Issue description

The `https-URI` ABNF production in Section 2.7.2 of RFC 7230 contains the same inconsistency with RFC 3986 identified for `http-URI` in erratum 4251, affecting how HTTPS URIs should be parsed and generated by conforming implementations.

# Errata 6809 - RFC 8986

- **RFC Title:** Segment Routing over IPv6 (SRv6) Network Programming
- **RFC Publication Date:** February 2021

```
Section 4.10 says:


When N receives a packet whose IPv6 DA is S and S is a local End.DX2

SID

// state how to fix the above text here

```

## Explanation

The original text uses an incorrect SID type. This inconsistency misidentifies the relevant SID type and may lead to incorrect processing of packets with such SIDs.
